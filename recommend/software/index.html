    专业软件推荐

    视频制作

    OBS Studio

    直播，或者录制视频

    剪映

    简单好用的剪视频软件

    Adobe Premiere（PR）

    复杂一点的剪视频软件

    Adobe After Effects（AE）

    做视频特效的软件

    编程开发

    Visual Code

    用来开发Web相关，更轻量化的项目
    优点：很轻量，启动速度快

    Visual Studio 2022

    用来开发游戏或桌面应用程序，这样中大型体量的项目
    优点：很踏实可靠，功能稳定健全

    GitHub Desktop

    用来管理项目版本
    优点：简洁好用

    CG绘画

    Adobe Photoshop

    用来图像编辑，或者用来绘画
    优点：功能强大，历史悠久
    缺点：在绘画功能方面，不是专门为绘画设计的，功能有点臃肿

    SAI2

    用来绘制日系风格插画
    优点：体积小，启动快，功能完整

    优动漫PAINT（CSP）

    用来画插画或者漫画，许多专业画师的选择
    优点：素材库丰富，更新状态活跃

    Aseprite

    用来绘制像素画，适合像素画师和独立游戏开发者
    优点：体积小，有许多为像素画和游戏开发的独特功能支持

    模型动画

    Blender

    用来做3D建模、动画的软件
    优点：开源免费，社区活跃，功能更新快

    Live2D

    用来做2D模型动画的软件，虚拟主播用的比较多

    Spine

    用来做2D模型动画的软件，游戏开发用的比较多

    音乐制作

    MouseScore

    用来写乐谱的软件，免费、好用

    Ableton Live 11

    用来制作音乐的软件

    FMOD Studio

    用于游戏音乐的解决方案

    FamiStudio

    用来制作8bit复古风格的音乐

    游戏开发引擎 / 框架

    Godot

    正在快速发展的游戏开发引擎
    优点：开源免费，轻量，简单易上手
    缺点：目前发展还比较早期，许多功能不够完善

    Unity

    目前使用人数最多的游戏开发引擎
    优点：功能丰富，各种资源丰富
    缺点：学习周期长

    Unreal Engine（UE）

    主要用于制作3A大作、画面更真实的游戏开发引擎
    优点：开源，蓝图无代码编辑
    缺点：代码语言为C++，门槛难度略高

    Mono Game（XNA / FNA）

    一个经典的游戏开发框架
    优点：开源免费，轻量
    缺点：只是一个框架，很多功能都没有，门槛难度略高
